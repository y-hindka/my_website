Professional Experience

Here, I am working as an Embedded Software Engineer to ensure code robustness.

I use low-level languages like C and VHDL to accomplish this.

I led efforts to design, implement, debug, and test significant improvements to radar tracking algorithms on a complex, multi-threaded C++ code base. Throughout the process, I coordinated with Engineering Fellows and Subject Matter Experts to ensure the quality of my work.

In addition, I implemented and tested multi-threaded C++ web socket code using the Boost Beast library to enable two-way communication between critical components. I also Created a REST API using C++ to retrieve system status and perform operations on the system.

As part of my contributions to testing, I developed MATLAB scripts to analyze simulation data, allowing my team to verify requirements were met. Additionally, I crafted full system BDD tests using Gherkin, Squish, and Python as well as C++ unit tests using the GoogleTest framework.

In my internship at Leidos, I was tasked with modernizing a management system for one of the company's warehouses.

I collaborated with a team to migrate an outdated SQL Server database to AWS and develop a streamlined website to access and modify the information in that database. The new website reduced redundancy within the legacy warehouse management system and provided a much more streamlined experience for the warehouse employees.

I used ReactJS to create a frontend system that used a REST API to interact with a Spring Boot backend.
